## Question Summary

### Question 1 – Linux Environment Verification

Verified the Linux environment by identifying the current user, groups, shell, working directory, workspace contents, and network connectivity.

### Question 2 – Secure Project Workspace Setup

Created a secure project workspace, configured permissions, verified ownership, checked the default umask, and documented how Linux permissions protect project data.

### Question 3 – File System and Link Analysis

Created an original file, a hard link, and a symbolic link. Compared their inode numbers, observed the effect of deleting the original file, and analyzed file metadata.

### Question 4 – File Access and I/O Investigation

Investigated Linux file I/O by identifying open files, examining file descriptors, demonstrating output and error redirection, and reviewing system resource limits.

### Question 5 – Storage Health Assessment

Collected storage information, examined mounted file systems, checked disk usage and inode utilization, and documented the findings using the `vi` editor.
