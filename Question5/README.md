# Question 5 - Storage Health Assessment

## Commands Executed

### Command 1

```bash
mkdir ~/Question5
cd ~/Question5
```

**Explanation:** Created a working directory for the storage assessment.

---

### Command 2

```bash
lsblk
```

**Explanation:** Displayed all available storage devices and partitions.

---

### Command 3

```bash
df -h
```

**Explanation:** Displayed mounted file systems and their disk usage.

---

### Command 4

```bash
du -sh .
```

**Explanation:** Displayed the disk space used by the current directory.

---

### Command 5

```bash
df -i
```

**Explanation:** Displayed inode utilization for mounted file systems.

---

### Command 6

```bash
vi Storage_Assessment_Report.txt
```

**Explanation:** Opened the `vi` editor to create and save the storage assessment report.
